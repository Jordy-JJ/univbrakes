# UNIVBRAKES - Sitio web

Paquete listo para publicarse con GitHub Pages.

## Publicar

1. Crea un repositorio nuevo en GitHub.
2. Sube `index.html`, `404.html` y `.nojekyll` a la raíz del repositorio.
3. En GitHub abre **Settings > Pages**.
4. En **Build and deployment**, selecciona **Deploy from a branch**.
5. Elige la rama `main` y la carpeta `/ (root)`.
6. Guarda y espera a que GitHub muestre la dirección pública.

El archivo `index.html` contiene las imágenes dentro del propio HTML, por lo que no necesita carpetas adicionales.
